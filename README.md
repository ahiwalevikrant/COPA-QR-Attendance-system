# 📱 COPA ITI - QR Attendance Management System

A modern, web-based attendance management system using QR code scanning. Built with vanilla JavaScript, HTML5, and CSS with a sleek glassmorphism UI design.

**Live Demo:** [GitHub Pages Link](https://ahiwalevikrant.github.io/COPA-QR-Attendance-system/)

---

## ✨ Features

### 🔐 Authentication
- Secure login screen with credential validation
- Session management
- User profile display in navbar
- Logout functionality

### 📱 QR Code Scanning
- Real-time camera-based QR code scanning
- HTML5 QRCode library integration
- Instant attendance marking
- Laser scan effect animation
- Scanner status indicators (Ready/Success)

### 📊 Dashboard & Analytics
- Real-time attendance statistics
- Total students, present, and absent counts
- Attendance trend charts (Chart.js)
- Recent scan activity log
- Visual progress indicators

### 📋 Attendance Management
- Detailed attendance records table
- Search and filter functionality
- Date range filtering
- Student information display
- Attendance status badges
- Responsive data grid

### 📥 Data Export
- **Excel Export** (XLSX format) - SheetJS library
- **PDF Export** (with tables) - jsPDF + AutoTable
- Date-filtered exports
- Formatted reports

### 🎓 Student Cards
- Dynamic QR code generation on student cards
- Student profile information
- Printable ID cards with embedded QR codes
- QRCode.js integration

### 💾 Data Persistence
- Local storage for attendance records
- Student database management
- Settings persistence
- No backend required
